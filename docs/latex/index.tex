\hypertarget{index_autotoc_md0}{}\doxysection{Usage}\label{index_autotoc_md0}
Send requests to a \mbox{\hyperlink{namespace_nimiq}{Nimiq}} node with {\ttfamily Nimiq\+Client} object.


\begin{DoxyCode}{0}
\DoxyCodeLine{ \{c\#\}}
\DoxyCodeLine{var config = new Config(}
\DoxyCodeLine{    scheme: "http",}
\DoxyCodeLine{    host: "127.0.0.1",}
\DoxyCodeLine{    port: 8648,}
\DoxyCodeLine{    user: "luna",}
\DoxyCodeLine{    password: "moon"}
\DoxyCodeLine{);}
\DoxyCodeLine{}
\DoxyCodeLine{var client = new NimiqClient(config)}
\end{DoxyCode}


Once the client have been set up, we can call the methods with the appropiate arguments to make requests to the \mbox{\hyperlink{namespace_nimiq}{Nimiq}} node.

When no {\ttfamily config} object is passed in the initialization it will use the default values in the \mbox{\hyperlink{namespace_nimiq}{Nimiq}} node.


\begin{DoxyCode}{0}
\DoxyCodeLine{ \{c\#\}}
\DoxyCodeLine{var client = NimiqClient();}
\DoxyCodeLine{}
\DoxyCodeLine{// make rpc call to get the block number}
\DoxyCodeLine{var blockNumber = await client.blockNumber();}
\DoxyCodeLine{}
\DoxyCodeLine{Console.WriteLine(blockNumber) // displays the block number, for example 748883}
\end{DoxyCode}
